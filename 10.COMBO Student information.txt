PART 1 — COMPLETE VIVA / ORAL QUESTIONS & ANSWERS

1. Basic Java Questions

Q1. What is Java?

Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems.


---

Q2. Why is Java platform independent?

Java source code is converted into bytecode, and JVM executes the bytecode on any operating system.


---

Q3. What is bytecode?

Bytecode is intermediate machine-independent code generated by Java compiler.

File extension:

.class


---

Q4. What is JVM?

JVM (Java Virtual Machine) executes Java bytecode.

Functions of JVM

Loads classes

Verifies bytecode

Allocates memory

Executes program



---

Q5. What is JDK?

JDK (Java Development Kit) is used to develop Java programs.

It contains:

Compiler

JRE

Debugging tools



---

Q6. What is JRE?

JRE (Java Runtime Environment) provides environment to run Java applications.


---

Q7. Difference between JDK, JRE, JVM?

JDK	JRE	JVM

Development kit	Runtime environment	Executes bytecode

2. OOP Questions

Q8. What is OOP?

OOP stands for Object-Oriented Programming.

Main concepts:

Class

Object

Inheritance

Polymorphism

Encapsulation

Abstraction



---

Q9. Why class is used?

Class is used to create blueprint for objects.

It groups:

Variables

Methods

Data



---

Q10. Why object is created?

Object is created to access non-static members.

Without object:

Non-static methods cannot execute.



---

Q11. What happens if object is not created?

Compiler error occurs while accessing non-static members.


---

Q12. What is inheritance?

Inheritance means acquiring properties of parent class into child class.


---

Q13. Why inheritance is not used in this program?

Because this program mainly focuses on JDBC database operations.


---

Q14. What is extends keyword?

extends is used to inherit parent class.

Example:

class B extends A


---

Q15. What happens if extends keyword is removed?

Inheritance relationship breaks.

Child class cannot access parent methods.


---

Q16. Can subclass access private members?

No.

Private members are accessible only inside same class.


---

Q17. Why Java does not support multiple inheritance using classes?

To avoid ambiguity problem called Diamond Problem.


---

Q18. What is polymorphism?

One method showing different behaviors.


---

Q19. Is polymorphism present in this program?

No direct polymorphism is present.


---

Q20. What is abstraction?

Hiding implementation details and showing only functionality.


---

Q21. Is abstraction present here?

Yes, JDBC internally uses abstraction.


---
