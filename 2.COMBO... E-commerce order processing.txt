### 3. What is a class?

A class is a **blueprint for creating objects**.

👉 In this program:


### 4. What is an object?

An object is an **instance of a class that occupies memory in heap**.

Example:

```java

```

* To group data + methods
* To represent real-world entity (product)
* To reuse code

* To access class members

* Without object, memory is not allocated

To take **runtime input from user (keyboard input)**.

### 8. Why methods are used?

* To divide logic
* To make code reusable

### 9. Why constructors are used?

To **initialize object automatically**.

### 10. Why private variables are used?

To implement **encapsulation and data hiding**.
