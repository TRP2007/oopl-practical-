# 🔴 A. BASIC CONCEPT QUESTIONS

---

### 1. What is OOP?

👉 Object-Oriented Programming is a programming approach based on **objects and classes**.

---

### 2. Why Java is called object-oriented?

👉 Because everything is implemented using classes and objects.

---

### 3. What is class?

👉 A blueprint for creating objects with variables and methods.

---

### 4. What is object?

👉 Instance of class that occupies memory in heap.

---

### 5. Why object creation is needed?

👉 Without object, no memory allocation happens for class members.

---

### 6. What is Scanner class?

👉 Predefined Java class used for user input.

---

### 7. Why Scanner is used?

👉 To take runtime input from keyboard.
