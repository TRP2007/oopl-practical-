import java.util.Scanner;

// Superclass
class Animal
{
    // Private data members
    private String animalName;
    private String animalType;

    // Public method to accept animal details
    public void getAnimalDetails(String name, String type)
    {
        animalName = name;
        animalType = type;
    }

    // Public method to display animal details
    public void displayAnimal()
    {
        System.out.println("Animal Name : " + animalName);
        System.out.println("Animal Type : " + animalType);
    }
}

// Subclass inheriting superclass
public class Dog extends Animal
{
    // Private data member
    private int age;

    // Public method to accept dog age
    public void getAge(int age)
    {
        this.age = age;
    }

    // Public method to display dog details
    public void displayDog()
    {
        System.out.println("Dog Age     : " + age + " Years");

        if(age >= 5)
        {
            System.out.println("Category    : Adult Dog");
        }
        else
        {
            System.out.println("Category    : Young Dog");
        }
    }

    public static void main(String args[])
    {
        Scanner sc = new Scanner(System.in);

        Dog d = new Dog();

        System.out.print("Enter Animal Name: ");
        String name = sc.nextLine();

        System.out.print("Enter Animal Type: ");
        String type = sc.nextLine();

        System.out.print("Enter Dog Age: ");
        int age = sc.nextInt();

        // Calling superclass method
        d.getAnimalDetails(name, type);

        // Calling subclass method
        d.getAge(age);

        System.out.println("\n===== DOG DETAILS =====");

        // Display methods
        d.displayAnimal();
        d.displayDog();

        sc.close();
    }
}

# 📘 1. Problem Statement (Simple Explanation)

The program demonstrates **Single Inheritance in Java** using an **Animal–Dog relationship**.

* The superclass `Animal` stores general information like animal name and type.
* The subclass `Dog` inherits properties of `Animal` and adds specific information like dog age and category (Adult/Young).
* The program takes user input and displays complete dog details using inheritance.

👉 Purpose:
To understand how **inheritance and data reuse** work in object-oriented programming.

# 📘 2. Concepts Used (Definitions with Simple Meaning)

## 🔹 Class

A class is a blueprint used to create objects.
👉 Example: `Animal`, `Dog`

---

## 🔹 Object

An object is a real instance of a class.
👉 Example: `Dog d = new Dog();`

---

## 🔹 Inheritance

It is a mechanism where one class acquires properties of another class.
👉 Example:
`Dog extends Animal`

✔ Advantage:

* Code reusability
* Easy maintenance

---

## 🔹 Superclass

The parent class from which properties are inherited.
👉 Example: `Animal`

---

## 🔹 Subclass

The child class that inherits properties from superclass.
👉 Example: `Dog`

---

## 🔹 Method

A block of code that performs a specific task.
👉 Example: `displayAnimal()`, `displayDog()`

---

## 🔹 private keyword

It restricts access to variables within the same class only.
👉 Example: `private String animalName;`

---

## 🔹 public keyword

It allows access from outside the class.

---

## 🔹 this keyword

Refers to the current object of the class.
👉 Example: `this.age = age;`

---

## 🔹 Scanner Class

Used to take input from the user.
👉 Example:

```java
Scanner sc = new Scanner(System.in);
```

---

## 🔹 String

A class used to store text data.
👉 Example: `"Dog", "Animal"`

---

## 🔹 static keyword

Used for memory management and allows main method execution without object creation.
