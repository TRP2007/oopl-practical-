import java.util.Scanner;

// Abstract class
abstract class Employee
{
    // Private data members
    private String name;
    private double salary;

    // Method to accept employee details
    public void getDetails(String name, double salary)
    {
        this.name = name;
        this.salary = salary;
    }

    // Getter method for name
    public String getName()
    {
        return name;
    }

    // Getter method for salary
    public double getSalary()
    {
        return salary;
    }

    // Abstract method
    public abstract void calculateBonus();
}

// Subclass extending abstract class
public class Manager extends Employee
{
    // Implementing abstract method
    public void calculateBonus()
    {
        double bonus = getSalary() * 0.10;

        System.out.println("\n===== EMPLOYEE DETAILS =====");
        System.out.println("Employee Name : " + getName());
        System.out.println("Salary        : " + getSalary());
        System.out.println("Bonus         : " + bonus);
    }

    public static void main(String args[])
    {
        Scanner sc = new Scanner(System.in);

        Manager m = new Manager();

        System.out.print("Enter Employee Name: ");
        String name = sc.nextLine();

        System.out.print("Enter Salary: ");
        double salary = sc.nextDouble();

        // Calling method of abstract class
        m.getDetails(name, salary);

        // Calling implemented abstract method
        m.calculateBonus();

        sc.close();
    }
}

# 🟦 **DEFINITIONS OF ALL CONCEPTS USED**

## 🔹 Class

A class is a blueprint used to create objects. It contains variables and methods.

---

## 🔹 Object

An object is an instance of a class used to access methods and variables.

---

## 🔹 Abstract Class

A class that cannot be instantiated and may contain abstract methods.

👉 Used to provide structure and force subclasses to implement specific behavior.

---

## 🔹 Abstract Method

A method without a body that must be implemented in the subclass.

---

## 🔹 Inheritance

A mechanism where one class acquires properties of another class.

Example:

```java
class Manager extends Employee
```

---

## 🔹 Method Overriding

Redefining a parent class method in the child class with the same signature.

---

## 🔹 Encapsulation

Binding data and methods together and restricting direct access using private variables.

---

## 🔹 Scanner Class

A built-in Java class used to take input from the user.

---

## 🔹 String

A class in Java used to store text (sequence of characters).

---

## 🔹 static

A keyword that allows a method to run without creating an object.

Used in:

```java
public static void main(String args[])
```

---

## 🔹 void

Means the method does not return any value.

---

## 🔹 new

Used to create an object in heap memory.

---

## 🔹 this

Refers to the current object of the class.
