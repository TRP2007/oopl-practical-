# 🔴 A. BASIC CONCEPT QUESTIONS

---

### 1. What is a class?

👉 A class is a blueprint used to create objects. It contains variables and methods.

---

### 2. What is an object?

👉 An object is an instance of a class used to access data and methods.

---

### 3. Why do we create objects?

👉 Because memory is allocated only when an object is created in heap.

---

### 4. What is Scanner?

👉 A predefined Java class used to take input from user.

---

### 5. Why Scanner is used?

👉 To read runtime input from keyboard.

---

### 6. What is String?

👉 A class used to store text data. It is immutable.

---

### 7. Why String is immutable?

👉 To improve security, memory efficiency, and caching in JVM.

---

# 🔴 B. OOP CONCEPT QUESTIONS

---

### 8. What is abstraction?

👉 Hiding implementation details and showing only functionality.

---

### 9. Why abstraction is used?

👉 To enforce structure and reduce complexity.

---

### 10. What is inheritance?

👉 Mechanism where one class acquires properties of another class.

---

### 11. Why extends keyword is used?

👉 To implement inheritance.

---

### 12. What is polymorphism?

👉 One method behaving in many forms (overriding in this program).

---

### 13. Why method overriding is used?

👉 To provide specific implementation in child class.

---

### 14. Difference between overriding and overloading?

👉 Overriding = same method, different class
👉 Overloading = same method, different parameters

---

# 🔴 C. ABSTRACT CLASS QUESTIONS

---

### 15. Why abstract class is used here?

👉 To define common structure and force subclass implementation.

---

### 16. Can we create object of abstract class?

👉 No, it is incomplete.

---

### 17. What happens if abstract method is not implemented?

👉 Program becomes abstract and cannot be executed.

---

### 18. Why public is used?

👉 To allow access from anywhere.

---

### 19. Why private is used?

👉 To protect data (encapsulation).

---

### 20. Why static is used in main method?

👉 JVM calls main method without object creation.

### 21. What if static is removed?

👉 JVM cannot start execution → error.

### 22. Why void is used?

👉 Because main method does not return value.

---

### 23. What is this keyword?

👉 Refers to current object.

### 24. What is new keyword?

👉 Creates object in heap memory.

---

### 25. Why extends is used instead of creating new methods?

👉 To reuse existing code.

# 🔴 E. TRICKY EXAMINER QUESTIONS

---

### 26. What happens if extends is removed?

👉 Inheritance breaks → cannot access parent class methods.

---

### 27. Can subclass access private variables?

👉 No, private members are not accessible directly.

---

### 28. What if superclass method becomes private?

👉 It cannot be overridden.

---

### 29. Can parent class call child method?

👉 No, only child can extend parent.

---

### 30. What if object is not created?

👉 No memory allocation → program cannot execute methods.

---

### 31. What if Scanner is not closed?

👉 Memory leak may occur.

---

### 32. What if main method signature changes?

👉 JVM will not recognize program entry point.

---

### 33. Why Java does not support multiple inheritance?

👉 To avoid ambiguity (diamond problem).

---

### 34. Why String is immutable?

👉 Security, thread safety, and caching.

---

---

### 35. Where is object stored?

👉 In heap memory.

---

### 36. Where are references stored?

👉 In stack memory.

---

### 37. What is stack memory?

👉 Stores method calls and local variables.

---

### 38. What is heap memory?

👉 Stores objects dynamically.

---

### 39. What is garbage collection?

👉 Automatic memory cleanup of unused objects.

---

### 40. What happens during runtime?

👉 JVM executes bytecode line by line.

# 🔴 G. JVM / JDK / JRE QUESTIONS

---

### 41. What is JVM?

---

### 42. What is JDK?

👉 Java Development Kit = tools + JRE + compiler.

---

### 43. What is JRE?

👉 Java Runtime Environment = JVM + libraries.

---

### 44. What is bytecode?

👉 Intermediate code generated by compiler.

---

### 45. What does compiler do?

👉 Converts .java → .class (bytecode)

### 47. Why getters are used?

---

### 48. Why encapsulation is important?
