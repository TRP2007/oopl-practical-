1. Basic Java Viva Questions

Q1. What is Java?

Java is a high-level, object-oriented, platform-independent programming language used to develop software applications.


---

Q2. Why is Java called platform independent?

Java code is converted into bytecode, which runs on any operating system using JVM.


---

Q3. What is OOP?

OOP stands for Object-Oriented Programming.

Main concepts:

Class

Object

Inheritance

Polymorphism

Abstraction

Encapsulation



---

Q4. What is JDBC?

JDBC (Java Database Connectivity) is an API used to connect Java programs with databases.


---

Q5. Why JDBC is used in this program?

JDBC is used to:

Connect Java with MySQL

Execute SQL queries

Store employee data



---

2. Class and Object Viva Questions

Q6. Why class is used?

Class is used as blueprint for creating objects.

It groups:

Variables

Methods

Data members


Example:

class EmployeeJDBC
{
}


---

Q7. Why object is created?

Object is created to access non-static methods and variables.


---

Q8. What happens if object is not created?

Non-static methods cannot be accessed.

Compiler error may occur:

non-static variable cannot be referenced


---

Q9. What is reference variable?

A variable storing address of object.

Example:

Scanner sc


---

Q10. Where are objects stored?

Objects are stored in Heap Memory.
